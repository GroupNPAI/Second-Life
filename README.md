# A technical-economic model for end-of-life management to prevent photovoltaic waste

## Overview

This repository provides the datasets and supporting materials for a technical-economic model developed to support planning of second-life reuse and end-of-life (EoL) management of residential photovoltaic (PV) modules in Brazil. 

The model integrates module degradation, residential electricity-demand growth, and utility-based replacement behavior to identify critical temporal milestones where PV systems remain technically functional but become economically or behaviorally prone to premature replacement.

Contains the exact values of the identified system milestones (A, E, and R).

The consumption behavior data were derived from the methodology described in Section 3 of the associated study. A single production configuration (Class A) was simulated under the three consumption growth scenarios to analyze the interaction between techno-economic dynamics and user behavior.

To generate Figure 4, a simulation was conducted to estimate:

Growth of installed capacity
Corresponding module mass (in tonnes)

In addition, an assumption was made that 23% of systems are subject to premature disposal, allowing evaluation of its potential impact on waste generation.

The dataset used for Figure 3 is provided in Sheet 3.

This analysis is based on historical data of grid-connected photovoltaic installations obtained from the Brazilian Distributed Generation Data Panel (PDGD), available at:
